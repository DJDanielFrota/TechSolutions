TechSolutions - Segurança & TI Profissional
Esta é uma Landing Page moderna e responsiva desenvolvida para a TechSolutions, focada em serviços de infraestrutura de TI, manutenção de hardware e sistemas de segurança (CFTV). O projeto foi construído com foco em conversão, design limpo e performance.

🚀 Funcionalidades
Design Responsivo: Adaptado para dispositivos móveis, tablets e desktops usando Tailwind CSS.

Formulário de Contato Inteligente: Integração com Formspree para envio de e-mails via AJAX (sem recarregar a página).

Botão Flutuante de WhatsApp: Acesso rápido para suporte e orçamentos via API do WhatsApp.

Navegação Suave: Scroll suave entre as seções da página.

SEO & Social Media: Tags Open Graph configuradas para compartilhamento otimizado em redes sociais.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica.

Tailwind CSS: Estilização utilitária e moderna via CDN.

JavaScript (ES6+): Lógica para envio de formulário e interatividade.

Font Awesome: Ícones vetoriais.

Google Fonts: Tipografia focada em legibilidade.

📧 Como funciona o Formulário?
O formulário utiliza o serviço Formspree. Para ativar o envio no seu próprio e-mail:

Crie uma conta em formspree.io.

Crie um novo formulário e copie o ID fornecido.

No arquivo index.html, localize a linha:

HTML
<form id="contato-form" action="https://formspree.io/f/xreybpky" method="POST">
Substitua xreybpky pelo seu ID pessoal.
